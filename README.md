## 🇺🇸 English

This project is a **custom UI component library** built with Angular using **Reactive Forms**.  
It provides reusable form components with validation, live preview, and code examples.

### 🚀 Features
- Reusable form components
- Angular Reactive Forms
- Validation (required, pattern, min, max)
- Tailwind CSS UI
- Live preview + code snippets

### 🧩 Components
- Text
- Email
- Password
- Number
- Checkbox
- Radio
- Select
- Textarea

### ⚙️ Installation
```
git clone https://github.com/AxelGMolaro/reactive-forms.git
cd reactive-forms
npm install
ng serve
```

### 🚀 Deployment
```
ng build --base-href=/reactive-forms/
npx angular-cli-ghpages --dir=dist/reactive-forms/browser
```
